PostgreSQL DBA

The PostgreSQL DBA training course is designed for users with existing DBA skills such as Oracle, MySQL or DB2. It provides the skills necessary to maintain PostgreSQL instances in a production environment. The course covers the following topics:

  • PostgreSQL installation and setup
  • Advanced PostgreSQL configuration
  • User and security management
  • Advanced PostgreSQL administration
  • Backup and restore
  • High availability
  • Advanced PostgreSQL monitoring
  • Query performance
