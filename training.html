Enhance your team's database expertise with DBA and developer training

Fujitsu runs interactive courses that will build on your team's database expertise, and tailors these courses to meet your specific requirements.

PostgresSQL DBA

The PostgreSQL DBA training course is designed for advanced users with existing DBA skills such as Oracle, MySQL or DB2. It provides the skills necessary to maintain PostgreSQL instances in a production environment. Fujitsu tailors these courses based on the customer's requirements and is therefore able to change the course emphasis if required.

The course covers the following topics:

  • PostgreSQL installation and setup

  • Advanced PostgreSQL configuration

  • User and security management

  • Advanced PostgreSQL administration

  • Backup and restore

  • High availability

  • Advanced PostgreSQL monitoring

  • Query performance

PostgreSQL Developer

The PostgreSQL Developer training course will benefit software developers who want to gain knowledge of PostgreSQL features, syntax and best practices.

The course covers the following topics:

  • Advanced SQL - includes PostgreSQL-specific language features, query optimization, query rewriter, rules, triggers, views, scan types, joins, sorts, aggregation, explain, indexes, statistics, common query problems, clustering and reindexing

  • PL/PgSQL procedural language - includes syntax and structure, data types, loops and constructs, exception handling, set returns and diagnostics

Replication and High Availability

The Replication and High Availability training course develops the skills of participants in the areas of replication and high availability.

The course covers the following topics:

  • Replication concepts

  • PITR backup and recovery

  • Warm and hot standby

  • Streaming replication

  • Slony

  • Load balancing using pgpool-II
