Features

FUJITSU Enterprise Postgres extends PostgreSQL with additional features such as:

  • One-click backup and recovery
  • Secure 256-bit transparent data encryption (PCI-DSS compliant)
  • Data masking
  • Parallel scanning and aggregation
  • In-Memory Column Store (IMCS) (*)
  • WAL duplication
  • DB mirroring controller
  • Enhanced Oracle® compatibility for simple migration
  • Enhanced GUI cluster management console
  • System usage statistics

* IMCS implementation by Fujitsu Laboratories Ltd

Contact us for more information on any of these features.
